About This Demo

This page demonstrates Web Bot Authentication verification using HTTP Message Signatures (RFC 9421). The verification endpoint expects Signature, Signature-Agent, and Signature-Input headers and verifies cryptographic signatures using the Ed25519 algorithm. Since browser requests don't include these headers, you'll see an error response by default.

API Access

This endpoint supports smart content negotiation. When you make a request with Accept: application/json header (e.g., using curl or any API client), you'll get a JSON response instead of this HTML page. This allows the same endpoint to serve both human-readable documentation and programmatic API responses. 

curl -H "Accept: application/json" \
     -H "Signature: sig1=:..." \
     -H "Signature-Input: sig1=..." \
     -H "Signature-Agent: https://chatgpt.com" \
     https://wba-quickstart.vercel.app
